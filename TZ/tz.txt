# ТЗ
## Обязанности каждой роли 
### ==admin-story==:
 админ:
- все, что может ментор
- добавлять менторов(менторы проходят модерацию), блокировать менторов
- добавлять таски(таски проходят модерацию), блокировать таски, удалять таски
на сайдбаре админа будет вкладка panel(на вкладке panel будут заяки менторов)
одобение и запросы более подробно информации будут запрашиваться через почту
инфа по одобрению или отклонению заявки будет дублироваться на почту
будет раздел с добавлением тасок 

### ==mentor-story==:
ментор может:
- все, что может юзер
- добавлять таски, редактировать таски, удалять таски, менять время выполнения таски
- общаться с учениками
на сайдбаре ментора будет располагаться графа редактирования таски


### ==user-story==:
юзер может: 
- просматривать доступные таски
- просматривать выполненные таски 
- общаться с менторами 
- выполнять таски и кидать их выполнение
- оценивать ментора

Таска будет иметь: 
- название
- обучающий видео-материал
- тег, описывающий направление таски
- описание
- чекпоинты

## Описание страниц

### ==Страница админ-панели(Admin-Panel):==
Элементы:
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пол-ля.
		- сайдбар - боковое меню, которое содержит ссылки для перехода на другие страницы.
		- верхняя панель с табами, содержащая 2 кнопки(mentors, admins, categories), которые переключают контентную часть
		- контентная часть - блок, содержащий контент выбранного раздела. В контентной части будут отображатся следующие компонеты:
			- Mentors - При нажатии на кнопку mentors появляется список заявок поль-лей на менторство. В 1 заявке содержится имя, адрес почты ментора, а также кнопка добавить или отклонить. При нажатии на кнопку добавить, юзер приобретает права ментора, на почту высылается электронное оповещение об этом, в панели уведомлений также появляется соответствующее сообщение. При нажатии на кнопку отклонить, появляется небольшое поле для текста причины отклонения заявки, на почту и в панель уведомлений также направляется соответствующее оповещение. Дальнейшие вопросы решаются через почту. 
			- Admins - при нажатии на кнопку admins manager появляется поле ввода для поиска юзеров, которые должны получить права администратора. В каждой записи полученного списка пользователей будет отображаться имя, почта, роль и кнопка сделать админом или лишить прав админа, если она есть. Любое действие с правами администратора(добавление, удаление) сопровождается письмом на почту. Любой пользователь, назначенный администратором, не будет иметь полномочий для добавления других администраторов.
			- Categories - при нажатии на кнопку categories будет появляться поле с поиском категории по названию, Кнопка для добавления категории, список всех категорий. В каждой строке категорий будет кнопка удаления категории. 
### ==Страница настроек(Settings):==
Элементы:
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пол-ля.
		- сайдбар - боковое меню, которое содержит список ссылок на другие страницы.
		- верхняя панель с табами для перехода между разделами настроек. Страницы с настройками:
			- Account: на данной странице содержится список пользовательских данных, доступных для изменения, а именно: имя, почта, фотография профиля.
			- General - на данной странице содержится список настроек приложения, а именно: возможность выбрать язык
			- Notifications - на данной странице содержится список настроек уведомлений, а именно: переключатель уведомлений о сообщениях, переключатель уведомлений о дедлайнах таски, переключатель уведомлений о новых тасках
		- контентная часть с настройками, переключаемая с помощью табов. Страницы с настройками:
			- Account: на данной странице содержится список пользовательских данных, доступных для изменения, а именно: имя, почта, фотография профиля.
			- General - на данной странице содержится список настроек приложения, а именно: возможность выбрать язык.
			- Notifications - на данной странице содержится список настроек уведомлений, а именно: переключатель уведомлений о сообщениях, переключатель уведомлений о дедлайнах таски, переключатель уведомлений о новых тасках.
## ==Страница сообщений(Message):==
Элементы:
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пользователя.
		- сайдбар - боковое меню, которое содержит список ссылок для перехода на другие страницы.
		- список чатов с другими людьми. Каждый чат представляет собой таб, содержащий фото другого пользователя, имя другого пользователя, фрагмент текста последнего сообщения, оставленного в чате, дату или время последнего сообщения, статус сообщения(новое сообщение(от другого пользователя), непрочитанное сообщение(отправленное текущим пользователем)).
		- Панель с данными другого участника диалога, которая содержит фото профиля пользователя и имя пользователя.
		- чат с другим пользователем. 
			- Чат представляет собой список сообщений, отличающихся цветом и расположением(сообщения текущего пользователя размещаются справа, другого пользователя слева). Чат включает в себя возможности отправки текстовых сообщений, файлов, медиафайлов. 
			- рядом каждым сообщением будет указано время отправки(ближе к середине), в верхней части чата будет указана дата переписки. 
			- В нижней части чата будет располагаться поле для ввода сообщения, кнопка для загрузки фалов, медиафайлов. При загрузке файлов, медиафайлов будет появляться панель с именем загружаемого файла и размером, а также кнопка для удаления прикрепленного файла, медиафайла. Фотографии и видеозаписи будут иметь окно небольшое окно предпросмотра.  

## ==Страница менторов Mentors:==
элементы: 
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пользователя.
		- сайдбар - боковое меню, которое содержит список ссылок для перехода на другие страницы.
		- Панель для поиска менторов. Компоненты панели для поиска менторов:
			- поле для поиска по имени. После ввода имени в строку поиска, слайдеры пропадают и появляется список с результатами запроса. Результат представляет собой список в виде карточек с информацией о менторах. Каждая карточка будет содержать фотографию профиля , имя, фрагмент описания профиля, категорию ментора и кнопку "Follow", количество созданных тасок, рейтинг, количество оценок. При нажатии на кнопку будет происходить редирект на страницу с информацией о профиле ментора.
			- кнопка категорий(category). При нажатии на кнопку будет появляться модальное окно со списком категорий и кнопкой применения сортировки. Каждое поле будет иметь название категории и чекбокс для выбора. 
			- кнопка сортировки(Sort By). При нажатии н кнопку будет появляться меню-бургер, которое содержит все типы сортировки. При нажатии на любой тип сортировки окно будет закрываться, будет показываться отсортированный список найденных записей менторов. Сортировка может происходить по следующим критериям:
				- Рейтинг ментора
				- Количество созданных тасок
		- заголовок для популярных менторов
		- горизонтальный слайдер со списком самых популярных менторов в виде карточек. Каждая карточка этого слайдера будет содержать имя, фото профиля, количество созданных тасок, рейтинг и количество оценок 
		- заголовок для менторов
		- cписок всех менторов. Каждая карточка списка будет содержать  фотографию профиля , имя, фрагмент описания профиля, категорию ментора и кнопку "Follow", количество созданных тасок, рейтинг, количество оценок.
			  
## ==Страница конкретного ментора:==
элементы:
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пользователя.
		- сайдбар - боковое меню, которое содержит список ссылок для перехода на другие страницы.
		- Контентная часть с информацией о менторе, а именно:
			- Фотография профиля ментора
			- Имя ментора
			- Категорию ментора
			- Кнопку "+Follow" или "Followed"
			- Описание аккаунта ментора
			- Рейтинг ментора 
			- Количество оценок 
			- слайдер, представленный в виде карточек. При нажатии на карточку происходит редирект на страницу с детальной информацией о таске. Каждая карточка содержит:
				- фоновую фотографию
				- название таски
				- тему таски
				- время выполнения 
				- количество приступивших к таске пользователей

## ==Страница тасок Tasks:==
элементы:
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пользователя.
		- сайдбар - боковое меню, которое содержит список ссылок для перехода на другие страницы.
		- панель поиска - предназначена для поиска тасок по нескольким критериям. Компоненты панели поиска:
			- поле ввода для поиска по названию таски. При вводе названия появляется список найденных тасок, представленный в виде карточек. Каждая карточка содержит:
				- фото (превью)
				- название таски
				- категорию таски
				- дату публикования таски
				- сколько человек выполнило таску
			- кнопка категорий. При нажатии на кнопку появляется модальное окно со списком доступных категорий. Каждая запись с категорией представлена в виде названия категории и чекбокса для выбора категории. После выбора категорий появляется список с найденными тасками представленный в виде каточек. Каждая карточка содержит:
			    - фото (превью)
				- название таски
				- категорию таски
				- дату публикования таски
				- фото профилей людей, которые выполнили таску
			- кнопка для сортировки тасок по нескольким критериям:
				- дедлайн взятых тасок
				- дата публикования тасок
		- контентная часть, содержащая разделы с тасками:
			- текущие таски пользователя. Каждая таска содержит:
			    - фото (превью)
				- название таски
				- категорию таски
				- прогрессбар таски(процент выполненной работа, который рассчитывается по чекпоинтам)
				- время выполнения таски
				- фото профилей людей, которые выполнили таску
			- новые таски. Каждая таска содержит:
			    - фото (превью)
				- название таски
				- категорию таски
				- время выполнения таски
				- фото профилей людей, которые выполнили таску
		- при сортировке или выборке по категориям основные разделы с карточками меняются на общий список с найденными карточками. 
## ==Страница с детальным описанием таски Detail Task:==
Элементы: 
		- навбар - верхняя часть страницы, которая содержит название страницы, на которой сейчас находится пользователь, кнопку для показа уведомлений, ссылку на страницу с настройками аккаунта, содержащую в себе фотографию профиля пользователя.
		- сайдбар - боковое меню, которое содержит список ссылок для перехода на другие страницы.
		- панель поиска - предназначена для поиска тасок по нескольким критериям. Компоненты панели поиска:
			- поле ввода для поиска по названию таски. При вводе названия появляется список найденных тасок, представленный в виде карточек. Каждая карточка содержит:
				- фото (превью)
				- название таски
				- категорию таски
				- дату публикования таски
				- сколько человек выполнило таску
			- кнопка категорий. При нажатии на кнопку появляется модальное окно со списком доступных категорий. Каждая запись с категорией представлена в виде названия категории и чекбокса для выбора категории. После выбора категорий появляется список с найденными тасками представленный в виде каточек. Каждая карточка содержит:
			    - фото (превью)
				- название таски
				- категорию таски
				- дату публикования таски
				- фото профилей людей, которые выполнили таску
			- кнопка для сортировки тасок по нескольким критериям:
				- дедлайн взятых тасок
				- дата публикования тасок
			- при сортировке или выборке по категориям основные разделы с карточками меняются на общий список с найденными карточками. 
		- контентная часть, которая содержит:
			- медиафайл
			- название таски
			- категории таски 
			- кнопка получить ментора
			- количество людей, выполнивших таску
			- время на выполнения таски
			- Описание таски
			- Чекпоинты для фиксации пройденных этапов (До 10 штук)
		- боковая панель с подробной информацией о выполненной работе, которая содержит:
			- заголовок панели
			- название таски
			- категории таски
			- заголовок о пользователе
			- имя пользователя с соответствующим полем
			- заголовок для файла
			- место для загрузки или перетаскивания файла
			- кнопка для отправки ответа
		- ответы отправляются на почту для проверки ментору, который создал таску. Работа считается выполненной, если пользователь отправил таску и уложился в срок
		
## ==Страница для редактирования тасок Edit Task(только для менторов и админов):==

Элементы:
		- табы для переключения контентной части. Каждый таб переключает контентную часть. Табы делятся на:
			- create, на странице будут следующие компоненты: 
				- заголовок
				- форма для создания таски:
					- Название таски
					- описание
					- Категория подставляется автоматически (как у ментора)
					- Время на выполнение таски
					- описание таски
					- обучающий медиаконтент
					- чекпоинты для фиксации процесса(до 10 штук)
					- кнопка для создания таски
			-
